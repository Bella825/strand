mm-footer

A fully-styled form footer - for use within a form footer or popover component. Capable of displaying variable button set and feedback messages.

Example HTML Editor

<mm-footer fitparent="true" unresolved> <mm-action underline="true"> <label>Close</label> </mm-action> <mm-button type="secondary"> <label>Save &amp; Close</label> </mm-button> <mm-button> <label>Save &amp; Add Another</label> </mm-button> </mm-footer> <!-- footer with error message --> <mm-footer id="footer" fitparent="true" unresolved> <mm-action underline="true"> <label>Close</label> </mm-action> <mm-button> <label>Save</label> </mm-button> </mm-footer> <!-- Script to display message --> <script> var footer; window.addEventListener("polymer-ready", function() { footer = document.body.querySelector('#footer'); footer.message = "Error some text is here."; footer.type = "error"; // show the message: footer.showMessage(); }); </script>

Example Output

Attributes

Attribute Options
message : String Default value: none The feedback message to display, usually feedback for validation success or failure. none
type : String Default value: info The type of inline message styling to be applied. 'error''warning''info''success'
fitparent : Boolean (Optional) Default value: null Instructs the component to fit to its parent container's width (percent or pixel width). none

Public Methods

Method Arguements
showMessage() Show the inline feedback message when needed. none
hideMessage() Hide the inline feedback message when needed. none

Public Properties

Properties Options
message : String Default value: none The feedback message to display, usually feedback for validation success or failure. none
type : String Default value: info The type of inline message styling to be applied. Valid values are ('error','warning','success' and 'info') none
